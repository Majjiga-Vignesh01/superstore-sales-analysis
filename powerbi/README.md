# Power BI Dashboard

This folder contains the Power BI dashboard files.

## Files

- dashboard.pbix  
  → Power BI project file (may not be uploaded due to size limits)

- dashboard.png  
  → Screenshot of the dashboard  

## Dashboard Features

- KPI cards (Total Sales, Profit, Orders)  
- Monthly sales trend  
- Region-wise performance  
- Category-wise analysis  
- Profit by sub-category  
- Discount vs Profit visualization  

The dashboard provides interactive insights into sales performance.
