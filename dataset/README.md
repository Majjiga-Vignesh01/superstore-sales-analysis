# Dataset

This folder contains the data used in this project.

## Files

- superstore_raw.csv  
  → Original dataset before cleaning  

- superstore_cleaned.csv  
  → Cleaned dataset used for analysis  

- superstore_cleaned.xlsx  
  → Excel file used for data cleaning process  

## Data Cleaning Steps

- Removed duplicate records  
- Handled missing values  
- Converted date formats  
- Ensured correct data types (Sales, Profit, Quantity)  
- Identified loss-making transactions  

The cleaned dataset is used for SQL analysis and dashboard creation.
